# Turbo Assistant

Unofficial JetBrains plugin that adds [Turborepo](https://turbo.build/) support — cross-file validation, navigation between `turbo.json` and `package.json`, run integration, and more — to WebStorm and IntelliJ Ultimate. Brings the kind of in-IDE awareness Turborepo deserves to the JetBrains ecosystem.

## Status

Pre-release. See [docs/v0.1-spec.md](docs/v0.1-spec.md) for the current milestone and [docs/roadmap.md](docs/roadmap.md) for what comes after.
