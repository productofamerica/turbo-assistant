# Architecture Decision Records

Short ADRs capturing significant architectural decisions for this plugin. One file per decision, named `NNNN-short-title.md` (e.g. `0001-use-jsonpsi-not-text.md`), where `NNNN` is a zero-padded sequence number that increments monotonically.

Two paragraphs each: the first states the decision and its context; the second lists the alternatives considered and why they were rejected. Keep them short — if a decision needs more than two paragraphs to explain, the decision itself is probably under-baked and should go back through brainstorming first.
